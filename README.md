# BookStoreClient

Фронтенд-приложение для интернет-магазина книг. Проект построен на современных веб-технологиях с фокусом на производительность, строгую типизацию и удобство пользовательского интерфейса.

## Стек технологий

- **Фреймворк:** [Next.js](https://nextjs.org/) (App Router)
- **Язык:** [TypeScript](https://www.typescriptlang.org/)
- **Стилизация:** [Tailwind CSS](https://tailwindcss.com/)
- **UI-компоненты:** [shadcn/ui](https://ui.shadcn.com/) (Стиль New York, базовый цвет Zinc)
- **Управление состоянием:** [Zustand](https://github.com/pmndrs/zustand) (для корзины и других глобальных состояний)
- **HTTP-клиент:** [Axios](https://axios-http.com/) (с настроенными интерцепторами для JWT-авторизации)

## Структура проекта

Ключевые директории:
- `/app` — маршрутизация и страницы приложения (Next.js App Router).
- `/components` — переиспользуемые React-компоненты.
  - `/components/ui` — базовые компоненты интерфейса от shadcn/ui (кнопки, модальные окна, карточки и т.д.).
- `/lib/api` — настройки для работы с сервером (например, инстанс Axios с обработкой токенов).
- `/store` — глобальные состояния приложения (например, `useCartStore` для корзины покупок).

## Установка и запуск

1. **Клонируйте репозиторий** (или скачайте исходный код).
2. **Установите зависимости:**
   ```bash
   npm install
   ```
3. **Запустите сервер для разработки:**
   ```bash
   npm run dev
   ```
4. Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере, чтобы увидеть результат.

## Интеграция с Backend

Приложение настроено на взаимодействие с бэкендом (предположительно на Spring Boot). 
- **Базовый URL API:** `http://localhost:8080/api`
- Все запросы, требующие авторизации, автоматически прикрепляют JWT-токен (если он есть в localStorage) к заголовку `Authorization: Bearer <token>`.

## Доступные скрипты (npm scripts)

- `npm run dev` — запуск приложения в режиме разработки.
- `npm run build` — сборка оптимизированного приложения для production.
- `npm run start` — запуск собранного production-приложения.
- `npm run lint` — запуск линтера ESLint для проверки кода.
